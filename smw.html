Social Media Widget

This web application is used to display on a page for an organization that has many social media pages. The Bootstrap framework is REQUIRED for this widget to be viewed correctly. Below is an interactive example of the widget (Facebook and Twitter are the only types that have entries in this example):


Facebook
Twitter
YouTube
Instagram
Select a Category:

If you have happend to click on Twitter, you have probably noticed that entries can have multiple categories. This helps by elimitating the confusion of designing the entries metadata.

Entries and categories can be created, editied and deleted with the Social Media Widget Manager. These tools are still in beta, but they work fairly well. Only minor issues are present in the latest versions.

Specification of media.txt File

IMPORTANT : You will not be able to create a media.txt in Microsoft Notepad as it does not follow the UNIX/OSX EOL format! Notepad++ or other editors that have an option to change the EOL format to UNIX/OSX will work. Please use the Social Media Widget Manager as it will automatically do this for you. None of the information below is nessesary to know if you use the manager.

This file has a certain specification that needs to be followed for the widget to work. It can be manually modified, but it is strongly recomeneded to use the Social Media Widget Manager to modify this file.

NEW

This starts every entry, including the categories. First, NEW MUST be added at the top of the file, followed by CATEGORIES along with two new lines.

CATEGORIES

This entry is only added ONCE in the top of the file. It specifies the names of each category that is used as metadata for each entry.

METADATA

Specifies the metadata associated with an entry. These are the Categories that are populated at the top of the file. When using in the CATEGORIES section, the categories follow the METADATA tag.

Social Media Tags

These specify a new entry in the widget. THEY MUST BE IN ALL CAPS AND FOLLOW THE NEW TAG!

After an entry has been created, the title is then entered followed by a new line. The link is then entered, followed by a new line. The METADATA tag is then entered and any multiples of categories can be entered, all followed by new lines. An example below shows a sample entry:

NEW
FACEBOOK
Facebook Developers
https://www.facebook.com/FacebookDevelopers
METADATA
Official Pages
