MP3 Organizer +

Do you have a folder full of MP3 files on your computer that you would like to be organized, but you dont have the time? This application is for you!

The MP3 Organizer + application organizes every single MP3 file in a directory, assuming that these files are tagged. It first creates a folder for the artist, then the album, then places the MP3 file in the respective directory.

This application is not ready yet, but will soon be released. I have decided to add some extra features that will take some time to implement.
